### Описание проекта

Данное исследование направлено на анализ факторов, определяющих субъективную оценку состояния экономики на индивидуальном уровне, а также на изучение расхождения между субъективными оценками и объективными макроэкономическими показателями.

### Основное внимание уделяется роли:

институционального доверия
восприятия функционирования демократии
ценностных и идеологических установок

Анализ проводится в сравнительной межстрановой перспективе для европейских стран.

Методология

### В исследовании используются:

линейные регрессионные модели (baseline)
модели градиентного бустинга:
LightGBM
XGBoost
метод интерпретации моделей:
SHAP (Shapley Additive Explanations)

Модели машинного обучения применяются для выявления нелинейных зависимостей и проверки устойчивости результатов.

### Основные результаты
Объясняющая способность модели возрастает с R² ≈ 0.47 (линейная регрессия)
до R² ≈ 0.51–0.52 (LightGBM и XGBoost)

### Наиболее значимые факторы:
удовлетворённость функционированием демократии
страновой контекст
институциональное доверие
восприятие качества государственных услуг

Объективные макроэкономические показатели оказывают ограниченное влияние
Социально-демографические характеристики играют второстепенную роль
Выявлена нелинейность влияния институционального доверия:
при низких значениях доверия эффект резко отрицательный
при переходе к средним значениям влияние быстро усиливается
при высоких значениях наблюдается эффект насыщения (≈ 6–7 баллов)
Не обнаружено «оптимального уровня» доверия — эффект остаётся положительным, но его прирост замедляется
Наблюдаются устойчивые межстрановые различия

### Исследовательские гипотезы
H1 — институциональное доверие положительно связано с оценкой экономики
H2 — эффект сохраняется после учёта институционального восприятия
H3 — существует расхождение между субъективными и объективными показателями
H4 — ценности модифицируют влияние доверия
H5 — социально-демографические факторы имеют ограниченный вклад
H6 — сохраняются межстрановые различия
